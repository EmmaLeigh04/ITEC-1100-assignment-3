Instructions


  1. Use Left/Right arrows to move the elf.

  2. Press Space to fix tree lights when you are close to the object.

  3. Fix all lights before Santa arrives (watch the shrinking timer bar!).

  4. If Santa arrives and lights are still broken → Game Over.
